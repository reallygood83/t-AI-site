간단한 사용 방법

1

Gemini API 키 발급

Google에서 무료로 제공하는 Gemini API 키를 발급받으세요.

2

API 키 입력

발급받은 API 키를 Teachers AI 서비스에 입력하세요.

3

AI 도구 사용
